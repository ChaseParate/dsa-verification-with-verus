# DSA Verification with Verus
## by Chase Harkcom

Sorting algorithm implementations formally verified using [Verus](https://github.com/verus-lang/verus).

## Algorithms
- Bubble sort
- Insertion sort
- Selection sort
- Merge sort

Each algorithm is verified to produce output that is both sorted and a permutation of the input.

## Running
Install Verus, and simply run `make`.
